\section{Rustc PR Tracking}
\label{sec:rustc_pr_tracking}

Rustc PR Tracking\footnote{\url{https://rust-lang.github.io/rustc-pr-tracking/}} je statická stránka zobrazující informace o~\PR z~repozitáře překladače Rustu. Nástroj je specifický pro repozitář rust-lang/rust a~závisí na jeho konkrétních štítcích a~procesním workflow.

\subsection{Obsah a~funkcionalita}
\label{subsec:rustc_pr_tracking:content}

Nástroj je primárně zaměřen na sledování stavu \PR a~aktivity v~nich. Kromě překladače Rustu monitoruje i~další repozitáře organizace rust-lang, konkrétně crates.io, rust-analyzer a~clippy. Crates.io je registr balíčků pro ekosystém jazyka Rust.

Zobrazovaná data lze filtrovat podle časového rozsahu, nejméně za poslední den, nejvýše za posledních 999 dní. Stránka se aktualizuje denně.

\subsection{Zhodnocení}
\label{subsec:rustc_pr_tracking:evaluation}

Jedná se o~jednoduchý statický web s~denní aktualizací dat. Nástroj je snadno pochopitelný a~přehledný. Poskytuje detailní vhled do stavu \PR, zejména co se týče čekání na code review nebo na sloučení.

Nevýhodou je úzké zaměření výhradně na \PR, issues ani celkové metriky repozitáře nejsou sledovány.
